# PhD

Repositorio para desenvolvimento de codigo do PhD em Deep Learning aplicado a monitorizacao com dados InSAR.

## Ambiente
- Conda: `miguelgpu`

## Estrutura
- `docs/` notas e roadmap
- `data/` dados (nao versionar `data/raw`)
- `configs/` configuracoes de pipelines e experimentos
- `pipelines/` wrappers por ferramenta InSAR
- `src/` codigo de preprocessing, features, modelos e avaliacao
- `notebooks/` exploracao e prototipagem
- `tests/` testes rapidos
- `env/` arquivos de ambiente
- `scripts/` utilitarios

## Quickstart
1. Ativar o ambiente: `conda activate miguelgpu`
2. Ver `docs/roadmap.md` para o plano de fases.
